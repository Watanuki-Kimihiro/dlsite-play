DLsite専用WEBアプリ DLsite Play

同人誌、電子書籍、音声・動画がすぐに楽しめるビューワーアプリ

Windows / Mac / iOS / Androidに対応
対応作品は160,000作品以上

今すぐはじめる

様々な作品形式をサポート

  • PDF
  • テキスト
  • 音声・音楽
  • CG集
  • 動画
  • HTML
同人誌・電子書籍
PDFファイル、テキストファイル
音声・音楽
FLAC、MP3、M4A、WAV、AAC、WMAなどのオーディオファイル
CG集
JPEG、GIF、BMP、PNGなどの画像ファイル
動画
AVI、WMV、MP4、FLV、MPEG、MOVなどの動画ファイル
その他
HTML作品

HTML作品

スマートフォンなどでHTML作品を読みやすく表示する、リーダーモードを搭載。

ライブラリ

ライブラリでは作品の検索・閲覧ができます。

ライブラリ

インクリメンタルサーチを採用し、1文字入力するごとに随時検索。作品を素早く探し出すことができます。また、検索は作品名だけでなくクリエイター名での検索も可能となっております。

ジャンル絞り込み

ジャンルは左右フリックで別のジャンルに切り替えることができます。ジャンル切り替え後にジャンルタブをタップすることでより詳細なジャンルの選択も可能です。

ビューワー

ブラウザ視聴対応作品なら、その場で閲覧することができます。

同人誌・電子書籍・CG集

見開きページ、自動ページ送りに対応。シークバーで目的のページまで飛ばすことも可能です。

音声・音楽

シャッフル、リピート、バックグラウンド再生に対応。音声を再生しながらライブラリを閲覧することも可能です。

動画

フルスクリーン再生に対応。

HTML作品

リーダーモード搭載。スマートフォンに最適化されたデザインで閲覧することができます。

ファイルのダウンロード

ファイルを端末にダウンロードすると、データ通信量を節約したり、オフラインで閲覧したりすることができます。

ダウンロード機能

作品を丸ごとダウンロードしたり、一部のファイルだけダウンロードしたりと柔軟な対応も可能です。
（ダウンロード可能なファイルは画像 / 音声 / テキスト）

オフライン利用

機内モードなど、インターネットに接続されていない状態でもDLsite Playを利用可能になりました。

ストレージ管理

ダウンロードしたファイルはストレージ管理から確認・削除ができます。

マイリスト・プレイリスト

お気に入りの作品・楽曲の再生リストを作成できる機能です。それぞれのページからの追加はもちろんのこと、作品詳細ページからも追加が可能です。

マイリスト

お気に入りの作品をブックマーク（お気に入り）として保存することができる機能です。

プレイリスト

お気に入りの楽曲でオリジナルの再生リストを作成できる機能です。プレイリストは同じファイルを複数追加することができます。

マルチデバイス対応

PC / スマートフォン / タブレットのすべてのデバイスでご利用いただけます。WEBアプリなので、すべてのプラットフォームで動作可能です。

クラウドでデータを共有

クラウドに対応しているので、異なるデバイス間でデータを共有することができます。

注意事項

  • DLsite Playの利用には、DLsiteへのユーザーアカウントでのログインが必要となります。
  • サークル様およびメーカー様が退会した作品、及び2015年以前に販売終了した作品は、DLsite Playでの閲覧ができません。
